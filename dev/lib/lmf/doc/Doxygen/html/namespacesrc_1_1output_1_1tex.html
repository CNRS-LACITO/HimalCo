Write a LaTeX file.

Parameters
objectThe LMF instance to convert into LaTeX output format.
filenameThe name of the LaTeX file to write with full path, for instance 'user/output.tex'.
preambleThe name of the LaTeX file with full path containing the LaTeX header of the document, for instance 'user/config/japhug.tex'. Deafult value is None.
lmf2texA Python dictionary describing the mapping between LMF representation and LaTeX commands. Default value is 'lmf_tex' dictionary defined in 'src/config/tex.py'. Please refer to it as an example.
orderA Python list defining the order in which LMF information must be written, for instance ["Lemma.lexeme", "LexicalEntry.partOfSpeech"]. Default value is 'tex_order' list defined in 'src/config/tex.py'.
